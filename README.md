# LAB4
## Requirements
- Node.js 16.x or higher (tested on v16.15.1)
## Getting Started
Clone repository:
```bash
git clone <this>
cd lab4
```
Start postgres with docker
```bash
docker-compose up
```
Install dependencies
```bash
cd server
npm install
cd ../
cd client
npm install
```
Start server
```bash
cd server
npm run start
```
Start server
```bash
cd client
npm run start
```
Default port for server is 3004 and for client 3000
